# Volunteer Tanggap Bencana
Fullstack project (Backend: Node.js + Express + MongoDB; Frontend: static HTML/CSS/JS).
Ready to deploy: Frontend -> Vercel; Backend -> Railway/Render.

## Quickstart (local)
- Backend:
  1. cd backend
  2. cp .env.example .env  -> edit MONGO_URI and JWT_SECRET
  3. npm install
  4. npm run dev

- Frontend:
  1. cd frontend
  2. Serve static files (or open index.html)
